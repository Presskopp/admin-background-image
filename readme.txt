Adds a customizable background image to the WordPress admin with live preview, overlay color, and blur.

== Description ==

Admin Background Image allows you to customize the look and feel of your WordPress admin area by adding a background image with optional overlay color and blur effects.

The plugin is lightweight, fast, and applies all changes instantly.

**Key Features:**

* Live background preview (no page reload required)
* Media Library integration for image selection
* Adjustable overlay opacity
* Custom overlay color picker
* Optional blur effect
* Instant removal and reset
* Automatic saving via AJAX
* Clean and minimal UI

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/admin-background-image/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **Appearance → Admin Background**

== Usage ==

1. Click **"Select Image"** to choose a background from the Media Library
2. Adjust overlay opacity and color
3. Optionally add blur
4. Changes are applied instantly and saved automatically

To remove the background:

* Click **"Remove"** to reset all settings

== Features ==

**Live Editing**

All changes are applied instantly without reloading the page.

**Overlay & Color**

Adjust overlay intensity and choose any color to match your admin UI.

**Blur Effect**

Add a blur layer for improved readability.

**Minimal UI**

Controls are only visible when a background image is set.

== Frequently Asked Questions ==

**Q: Does this affect the frontend?**
A: No, the plugin only modifies the WordPress admin area.

**Q: Can I use any image size?**
A: Yes. The image is stretched to fill the screen, so using an appropriate size is recommended. You’ll see the result instantly.

**Q: Will this slow down my admin area?**
A: No, the plugin is lightweight and optimized for performance.
